## Use
* Android Remote Service
* Content Observer (FileObserver not working on 6.0)


## Output
* orginal screenshot file will be removed.
* new file name will be created.

## Android version
API14 ~ Recent
tested : GalaxyS2, GalaxyS3, LG G4 etc..
